TryHackMe’s Newest Learning Path, A Game Changer For Cyber Security Beginners

2021-07-08 :: by DODEA

TryHackMe: https://tryhackme.com/

Pre-Security Learning Path: https://tryhackme.com/path/outline/presecurity

The Pre-Security learning path is the newest addition to TryHackMe’s range of learning paths, which if you are unfamilar are slef-paced courses that teach you about a particular area of cyber security. The Pre-Security learning path acts as a way for anyone brand new to IT and cyber security to learn the very basic pre-requesite knowledge to get them started on their cyber journey.

Pre-Security Learning Path Structure⌗

The Pre-Security learning path contains five sections including Cyber Security Introduction, Networking Fundamentals, How The Web Works, Linux Fundamentals and Windows Fundamentals. Each of these sections is then broken down further into serveral rooms, which are mini interactive classrooms, that teach one of the core topics related to the section. As you complete each room the content builds on what you learn’t from the last one until you have a good understanding of the fundamental concepts for the seciotn. For example in the Networking Fundamentals section there are five total rooms that starts from the very beginning of what networking is and slowly builds up the participants knowledge until they have a solid introductory knowledge of networking. Below is a screenshot showing TryHackMe’s learning path structure for the Networking Fundamentals section.

As you can see from the image above the section called Networking Fundamentals is broken down into rooms which each teach a unique aspect of Networking. By using this structure for teaching participants can easily see where the contents of each section leads and how each room is related giving them an understanding of why they are learning the provided content.

Cyber Security Introduction Section⌗

The Cyber Security Introduction section is the first section that the participant must work through and provides only a single room. This room gives an overview of why it is important to learn how the internet works and how networking works by providing a set of questions and an accompanying interactive activity for each part of the room. I really liked this aspect of the Pre-Security learning path because unlkie other cyber secuirty courses this one told you the why it is important to learn the provided content of this path. Further, by using a case study of a real life cyber attack on a well known company such as Target it shows particpants right from the start that what they are learning is applciable to real life. After that introduction into why these concepts must be learn’t as a cyber security professional the room then explains what you can do once you have finished the Pre-Security learning path giving participants some much need direction of where they can continue their cyber security training.

Networking Fundamentals Section⌗

The Networking Fundamentals section is the first content based section of the Pre-Security learning path and is one of the longest. This section covers so much of the foundational information about networking right from the basics of defining a network and the OSI model all the way down to packets and frames. However, the beauty of this section is that it strikes the right balance between reading and understanding theory and having some hands on pratical experiences. These pratical experiences were really what made this section so good because it allows you to see these netowrking concepts in acion through a visual hands on experience. For example one of my favourties was the Network Simulator, which can be seen in the image below, which allowed you to visualise how hosts can communicate on a network and how a TCP three way handshake is done in practice.

The only thing I would change about the Network Simulator would be the ability to pause the simulation and the ability to slow down the rate at which packets are travelling. I found that some times it was hard to keep up with what was going on and it would have been nice to slow it down or even pause it so you have enough time to process what is happening. However, despite the little improvement that could be made to the Network Simulator I found that on a whole these practical experiences were fantastic and something I wish I had when I first learn’t networking. I found this section so helpful despite alreay knowing a fair amount of networking and stronlgy believe that this section is a great tool for both beginners who want to learn the basics of networking and intermediate particpants that want to refresh their networking knowledge.

How The Web Works Section⌗

The How The Web Works Section continues on from the Networking Fundamentals section and starts to teach participants how the networking concepts relate to the internet. This section teaches content such as how internet services are implemented and accessed, how the HTTP protocol os used by web servers, how websites are made and then how all these concepts work together to prodivde us with the internet as we know it. Again I think this section strikes the perfect balance between theory and getting to interact with hands on praticals which makes for an enjoyable study experience.
